# Data Download Instructions

This directory holds raw and processed data files.
**Raw data is not committed to the repository due to size.** Follow the instructions below to obtain and place the data.

---

## 1. ERCOT Hourly Load Data

**Source:** Electric Reliability Council of Texas (ERCOT)
**URL:** https://www.ercot.com/gridinfo/load/load_hist

### Download Steps

1. Go to https://www.ercot.com/gridinfo/load/load_hist
2. Download the **Native Load** files for each year you need:
   - `native_load_2018.zip`
   - `native_load_2019.zip`
   - тАж
   - `native_load_2025.zip`
3. Extract each ZIP. Each file contains an Excel workbook (`.xlsx`) or CSV with columns:

   | Column            | Description                          |
   |-------------------|--------------------------------------|
   | `HourEnding`      | Timestamp (CST), format `MM/DD/YYYY HH:MM` |
   | `ERCOT`           | Total system load (MW)               |
   | (other zones)     | Zonal load тАФ not required by default |

4. Combine all years into a single CSV and save as:
   ```
   data/raw/ercot_load_2018_2025.csv
   ```

   **Minimum required columns:** `HourEnding`, `ERCOT`

### Quick Python Merge Script

```python
import pandas as pd, glob, pathlib

files = sorted(glob.glob("data/raw/native_load_*.csv"))
dfs = []
for f in files:
    df = pd.read_csv(f, usecols=["HourEnding", "ERCOT"])
    dfs.append(df)

combined = pd.concat(dfs, ignore_index=True)
combined.to_csv("data/raw/ercot_load_2018_2025.csv", index=False)
print(f"Saved {len(combined):,} rows.")
```

**Expected row count:** ~70,128 (8 years ├Ч 8,766 hours/year average)

## 2. ASOS Meteorological Data (Texas Stations)

**Source:** Iowa Environmental Mesonet (IEM) тАФ NOAA ASOS archive
**URL:** https://mesonet.agron.iastate.edu/request/download.phtml?network=TX_ASOS#

The paper uses three Texas ASOS stations:

| Station ID | Location             | Approx. Coordinates |
|------------|----------------------|----------------------|
| BKS        | Lubbock/Slaton, TX   | 33.5┬░N, 100.1┬░W     |
| JDD        | Jacksboro, TX        | 33.2┬░N, 98.2┬░W      |
| TME        | Houston/Taylor Lake  | 29.6┬░N, 95.1┬░W      |

### Download Steps

1. Visit https://mesonet.agron.iastate.edu/request/download.phtml?network=TX_ASOS#
2. For **each station** (BKS, JDD, TME):
   - Select the station from the dropdown
   - Set **Date range**: `2018-01-01` to `2025-12-31`
   - Select data variables:
     - `Air Temperature [F]` тЖТ `tmpf`
     - `Relative Humidity [%]` тЖТ `relh`
     - `Wind Speed [knots]` тЖТ `sknt`
     - `1 hour precipitation [inches]` тЖТ `p01i`
     - `Present Weather Codes` тЖТ `wxcodes`
   - **Data format:** Comma Delimited (CSV)
   - **Timezone:** UTC
   - Click **Get Data** and save as:
     ```
     data/raw/asos_BKS.csv
     data/raw/asos_JDD.csv
     data/raw/asos_TME.csv
     ```

### Alternative: IEM API (Automated Download)

```python
import requests, pathlib

STATIONS = {"BKS": "BKS", "JDD": "JDD", "TME": "TME"}
START, END = "2018-01-01", "2025-12-31"

for name, sid in STATIONS.items():
    url = (
        f"https://mesonet.agron.iastate.edu/cgi-bin/request/asos.py"
        f"?station={sid}&data=tmpf&data=relh&data=sknt&data=p01i&data=wxcodes"
        f"&year1=2018&month1=1&day1=1&year2=2025&month2=12&day2=31"
        f"&tz=UTC&format=comma&latlon=no&elev=no&missing=M&trace=T&direct=no&report_type=3"
    )
    resp = requests.get(url, timeout=120)
    out = pathlib.Path(f"data/raw/asos_{name}.csv")
    out.write_text(resp.text)
    print(f"Saved {out} ({len(resp.text):,} bytes)")
```

## 4. Data Quality Notes

- **Missing weather values:** The paper reports <0.6% missing; these are filled by linear time interpolation in `src/preprocessing/data_pipeline.py`.
- **ERCOT DST handling:** ERCOT timestamps are in CST/CDT (local time). The pipeline converts them to UTC using `pd.to_datetime(..., utc=True)`.
- **Multi-station averaging:** Meteorological variables from the three ASOS stations are averaged hourly to produce a representative Texas grid-wide weather signal.
